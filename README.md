# devTinder

Full-stack developer networking app built with Node.js, Express, MongoDB & React. 
Developers can discover and connect with each other through a swipe-based feed. 
Features include JWT authentication, connection requests, feed API with pagination, 
and profile management.

Built during Namaste Node.js course by Akshay Saini.

## Tech Stack
- Node.js & Express.js
- MongoDB & Mongoose
- React.js
- JWT Authentication
